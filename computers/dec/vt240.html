DECterm - VT 240

VT 240

The VT240 is a graphics terminal made by DEC. It supports ReGIS vector graphics and emulates a Tektronix 4010/4014. My VT240 has a green phosphor monochrome monitor.
It seems that the case has been painted. The plastic underneath the paint is noticeably yellowed. I'm not sure if it came from DEC this way or if someone painted it in the past. I'm assuming it came from DEC like this, as I found it in a closet at work.


ReGIS Graphics

I wrote a python script to convert an svg file to ReGIS commands to demonstrate the graphics capabilities of the terminal.
