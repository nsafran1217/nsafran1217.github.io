# C8000
*Last Updated: 02-May-2025*

The C8000 was HP's last PA-RISC workstation. All later PA-RISC systems lacked graphical consoles. 
This system features a single PA-8900 1.1 GHz dual-core processor.

More information can be found here: <a href="https://www.openpa.net/systems/hp_c8000.html" target="_blank" rel="noopener noreferrer">HP C8000 on OpenPA.</a>
I have HP-UX 11.11 (11i v1) TCOE installed, as it is the latest version that supports the C8000. I've been working on getting pkgsrc running but haven't had much success yet. 
HP-UX 11 has proven to be very particular about compilers and linkers, and I have yet to find a reliable way to compile software for the system.  
I'm sure its just something I'm doing wrong.
