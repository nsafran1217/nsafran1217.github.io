# SPARCStation5
*Last Updated: 02-May-2025*

This is another pizza box workstation, equipped with 32MB of RAM and a ~60MHz MicroSPARC
processor. Unfortunately, the memory in this system is proprietary and relatively rare,
making upgrades difficult. The prices for RAM on eBay don’t seem worth it for an
upgrade.
 
This system is running Solaris 2.6. I've installed a decent number of modern tools,
including gzip, bash, wget, etc.
