Terminal - Z80 LAIR

Z80 Logic Analyzer Interface Board

When I was first getting started with writing assembly, I found it very difficult to know what was happening after I burned the ROM and turned the comptuer on.
I purchased an Agilent 1670G to connect to the comptuer and get some visability into what the computer was doing. There is an inverse z80 assembler avaliable for the 1670G, which make it very easy to decode what the computer is doing.
This board is a clone of the LAIR Pro for the RC-2014 bus. I only changed the bus connection to fit my 40 pin layout.
