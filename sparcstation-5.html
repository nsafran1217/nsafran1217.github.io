SPARCStation 5

Sun Microsystems Home

Overview

This is another pizza box workstation. This one has 32MB of RAM and a 60-ish MHz MicroSPARC Processor. Unfortunately, the memory in this system is proprietary and relatively rare. The ebay price for it is too much for me.

This system is running Solaris 2.6. I've installed a decent number of modern tools, including gzip, bash, wget, and more.
